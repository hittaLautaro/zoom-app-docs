Moodle Zoom Meeting Client - Documentation

Overview

Moodle Zoom Meeting Client is a desktop application that enables seamless participation in Zoom meetings directly from Moodle LMS.

Installation

Windows

  1. Download MoodleZoomClient-Setup.exe
  2. Run the installer
  3. Follow the installation wizard
  4. Launch the application from Start Menu

macOS

  1. Download MoodleZoomClient.dmg
  2. Open the DMG file
  3. Drag the app to Applications folder
  4. Launch from Applications

Linux

  1. Download MoodleZoomClient.AppImage
  2. Make it executable: chmod +x MoodleZoomClient.AppImage
  3. Run: ./MoodleZoomClient.AppImage

How to Use

Step 1: Get Meeting Credentials from Moodle

  1. Log into your Moodle course
  2. Navigate to the Zoom meeting activity
  3. Copy the Meeting ID and Password

Step 2: Join the Meeting

  1. Launch Moodle Zoom Meeting Client
  2. Enter the Meeting ID
  3. Enter the Password
  4. Enter your display name
  5. Click "Join Meeting"

Step 3: During the Meeting

  • Use standard Zoom meeting controls
  • Mute/unmute microphone
  • Start/stop video
  • View shared screens
  • Use chat features

Troubleshooting

"Unable to connect" error

  • Verify meeting ID and password are correct
  • Check internet connection
  • Ensure meeting is active and hasn't ended

Audio/video not working

  • Check system permissions for camera and microphone
  • Verify device drivers are up to date
  • Test devices in system settings

App crashes on launch

  • Update to latest version
  • Reinstall the application
  • Contact support if issue persists

Uninstallation

Windows

  1. Go to Settings → Apps
  2. Find "Moodle Zoom Meeting Client"
  3. Click Uninstall

macOS

  1. Open Applications folder
  2. Drag app to Trash
  3. Empty Trash

Linux

Simply delete the AppImage file

Removing the Zoom App Authorization

  1. Log into Zoom account at https://zoom.us
  2. Go to Settings → Apps
  3. Find "Moodle Zoom Meeting Client"
  4. Click "Remove"

Note: Since this app uses anonymous join, most users won't need to perform this step.

Privacy & Data

This application does not store any personal data. All meeting credentials are used only in memory and discarded when the session ends. See our Privacy Policy for details.
